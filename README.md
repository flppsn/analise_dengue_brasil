# 🦟 Análise de Casos de Dengue no Brasil (2013–2023)

Análise exploratória dos casos reportados de dengue no Brasil,
cruzando dados epidemiológicos com informações populacionais (IBGE)
e climáticas (NASA Power).

## 📊 O que foi analisado
- Evolução temporal dos casos por ano
- Distribuição geográfica por estado (absoluta e por 100k habitantes)
- Relação entre precipitação e incidência da doença
- Mapa coroplético com escala logarítmica

## 🛠️ Tecnologias utilizadas
Python, Pandas, Seaborn, Matplotlib, Geopandas, Sidrapy, Requests

## 📁 Fonte dos dados
- Base de casos: FIAP
- População: IBGE/SIDRA (Censo 2022)
- Precipitação: NASA Power API
